Simple checkbox component with "indeterminate" status support.

### Installation

```bash
npm install --save react-indeterminate-checkbox
```

### Usage

```jsx
<IndeterminableCheckbox
   change={val => console.log(val)}
   name="mycheckbox-name"
   indeterminate
/>
```
